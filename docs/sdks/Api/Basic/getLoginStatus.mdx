# getLoginStatus

## 功能介绍

:::info 说明

获取用户登录状态

:::

:::caution 注意

注意连接状态和登录状态的区别，如果是未登录状态，可以调用login，其他状态不要调用login

:::

<TabItem value="iOS">

### 函数原型

```swift showLineNumbers
- (NSInteger)getLoginStatus;
```

### 输入参数

无

### 返回结果

| 参数名称 | 参数类型 | 描述    |
| -------- | -------- | ------- |
| NSInteger | status   | 登录状态，101：成功 102：登录中 103：登录失败 201：退出 |

### 代码示例

```swift showLineNumbers

NSInteger status = [OIMManager.manager getLoginStatus];

```

### 参数详解

| 参数名称    | 参数类型 | 是否必填 | 描述                                 |

| operationID | string   | 是       | 唯一随机字符串，用于根据日志定位问题 |

