# getFriendList

## 功能介绍

:::info 说明

（1）获取指定好友的昵称、头像等IM中的资料。  
（2）建议一次最大100个，因为数量过多可能会导致数据包太大被后台拒绝。

:::

### 函数原型

```swift showLineNumbers

- (void)getFriendListWithOnSuccess:(nullable OIMFullUsersInfoCallback)onSuccess
                         onFailure:(nullable OIMFailureCallback)onFailure;

```

### 输入参数

无

### 返回结果

| 参数名称  | 参数类型                                                                | 描述     |
| --------- | ----------------------------------------------------------------------- | -------- |
| onSuccess | [OIMFullUsersInfoCallback](docs/sdks/Callback/FullUserInfoCallback.mdx) | 成功返回 |
| onFailure | [OIMFailureCallback](docs/sdks/Callback/ErrorCallback.mdx)              | 失败返回 |

### 代码示例

```swift showLineNumbers

[OIMManager.manager getFriendListWithOnSuccess:^(NSArray<OIMFullUserInfo *> * _Nullable userInfos) {
} onFailure:^(NSInteger code, NSString * _Nullable msg) {
}];

```
