### SearchGroupMembersParam

| 字段名称               | 字段类型              | 描述                                           |
| ---------------------- | --------------------- | ---------------------------------------------- |
| groupID                | NSString              | 群 ID                                          |
| keywordList            | NSArray< NSString \*> | 搜索关键词，目前仅支持一个关键词搜索，不能为空 |
| isSearchUserID         | BOOL                  | 是否以关键词搜索 UserID                        |
| isSearchMemberNickname | BOOL                  | 是否以关键词搜索昵称，默认 false               |
| offset                 | NSInteger             | 偏移量                                         |
| count                  | NSInteger             | 数量                                           |
