### OIMSearchGroupParam

| 字段名称          | 字段类型              | 描述                                                          |
| ----------------- | --------------------- | ------------------------------------------------------------- |
| keywordList       | NSArray< NSString \*> | 搜索关键词，目前仅支持一个关键词搜索，不能为空                |
| isSearchGroupID   | BOOL                  | 是否以关键词搜索群 ID(注：两个不可以同时为 false)，默认 false |
| isSearchGroupName | BOOL                  | 是否以关键词搜索群名字，默认 false                            |
