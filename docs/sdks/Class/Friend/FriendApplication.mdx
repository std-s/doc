### OIMFriendApplication

| 字段名称      | 字段类型  | 描述                             |
| ------------- | --------- | -------------------------------- |
| fromUserID    | NSString  | 发起者 UserID                    |
| fromNickname  | NSString  | 发起者昵称                       |
| fromFaceURL   | NSString  | 发起者头像链接                   |
| toUserID      | NSString  | 被添加者 UserID                  |
| toNickname    | NSString  | 被添加者昵称                     |
| toFaceURL     | NSInteger | 被添加者头像链接                 |
| handleResult  | NSInteger | 处理结果（同意/拒绝/未处理）     |
| reqMsg        | NSString  | 申请添加好友时附带的消息         |
| createTime    | NSInteger | 创建时间                         |
| handlerUserID | NSString  | 处理者 UserID                    |
| handleMsg     | NSString  | 处理好友申请时附带的消息         |
| handleTime    | NSInteger | 处理时间                         |
| ex            | NSString  | 应用层扩展字段                   |
| attachedInfo  | NSString  | SDK 内部扩展字段，应用层无需关注 |
