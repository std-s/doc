### OIMBlackInfo

| 字段名称       | 字段类型  | 描述                             |
| -------------- | --------- | -------------------------------- |
| ownerUserID    | NSString  | 本人的 UserID                    |
| blockUserID    | NSString  | 黑名单 UserID                    |
| nickname       | NSString  | 黑名单的昵称                     |
| faceURL        | NSString  | 头像链接                         |
| createTime     | NSInteger | 拉黑时间                         |
| addSource      | addSource | 拉黑方式                         |
| operatorUserID | NSString  | 使之成为黑名单的 UesrID          |
| attachedInfo   | NSString  | SDK 内部扩展字段，应用层无需关注 |
| ex             | NSString  | 应用层扩展字段                   |
